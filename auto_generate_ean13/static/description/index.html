Auto Generate Product Barcode (EAN13)


Work With Both Community And Enterprise Version.

This application is generating product EAN13 barcode automatically (on product creation) or manually.
EAN13 barcode can be created by using a random number or using current date, you can add some prefix to the barcode, you have an option to auto-generate barcode on product creation, the barcode can be created in different sizes and colors. All setting saved per company, so different barcode can be created for the different company.
You can create barcode in bulk for already created products, you have the option to select multiple products and create barcode or select multiple product categories and create the barcode for all product of the selected product category.
Key features:
  • Easy To Use
  • Auto generate EAN13 barcode using random number.
  • Auto generate EAN13 barcode using current date.
  • Auto generate EAN13 barcode in using some prefix.
  • Auto generate EAN13 barcode in different size.
  • Auto generate EAN13 barcode in different colors.
  • Manage EAN13 barcode font size, Space before and after, text below barcode, distance between them
  • Auto generate EAN13 bulk barcode by selecting multiple products.
  • Auto generate EAN13 bulk barcode by selecting multiple products categories.
  • Company wise barcode generation setting.
  • Product can be search by ean13 barcode.
  • Set access right, so only authorized user will set Barcode.
* Configuration *
Go to Settings --> Companies --> Companies --> Select Any Company --> Configuration Page --> EAN13 Barcode Section.
To enable barcode generation functionality, you have to select Allow Barcode Generation access from setting --> users --> users --> Assess Right --> Other
Click on (Generate Barcode) button link form form view of product to manually generate EAN13 barcode for single product.
If Barcode is already set and you want to just set barcode image still click on(Generate Barcode) button link form form view.
If Prefix is set into company configuration and barcode create method is Generate EAN13 Using Random Number then in barcode first four digit are prefix that we set into company, next eight digit are any random number and last digit is checksum to validate ean13.
If Prefix is set into company configuration and barcode create method is Generate EAN13 Using Current Date then in barcode first four digit are prefix that we set into company, next two digit are day number, next two digit are hours, next two digit are minutes, next two digit are seconds and last digit is checksum to validate ean13.
If Prefix is set not set into company configuration and barcode create method is Generate EAN13 Using Random Number then in barcode first twelve digit are any random number and last digit is checksum to validate ean13.
If Prefix is not set into company configuration and barcode create method is Generate EAN13 Using Current Date then in barcode first two digit are year number, next two digit are month number, next two digit are day number, next two digit are hours, next two digit are minutes, next two digit are seconds and last digit is checksum to validate ean13.
Barcode can be generated in different Height, Width, colors, different Spacing.
Auto generate EAN13 bulk barcode by selecting multiple products. In this wizard you can override existing barcode by new barcode.
Auto generate EAN13 bulk barcode by selecting multiple products categories. In this wizard you can override existing barcode by new barcode. You can generate barcode for selected product categories and its child categories.
Generate ean13 barcode in list view.
Search product by ean13 barcode.

Installation Guide

sudo pip install pyBarcode

If you use version is python3 or above use below command

sudo pip3 install python-barcode
